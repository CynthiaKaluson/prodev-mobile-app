# ProDev Mobile App - Milestone 0x04

## Stack Navigation Implementation

This project implements stack navigation patterns for a React Native mobile application using Expo.

### Project Structure
- `app/` - Screen components and navigation layout
- `styles/` - Styling files separated from screen logic
- `constants/` - Application constants and asset references
- `assets/images/` - Image assets for the application

### Screens
- **Welcome Screen** - Landing page with navigation options
- **Sign In Screen** - User authentication form
- **Join Screen** - User registration form

### Navigation
Stack navigation pattern implemented with seamless screen transitions between welcome, sign-in, and join screens.

### ALX Requirements
All required files are present as specified in the task instructions.
